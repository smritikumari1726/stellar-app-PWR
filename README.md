# stellar-app-PWR
# 🛡️ Product Warranty Registry (Blockchain)

A decentralized application (dApp) built on **Stellar Soroban** that enables secure, tamper-proof product warranty management using blockchain technology.

---

## 🚀 Project Description

**Product Warranty Registry** is a blockchain-based system that allows users to register products, verify warranty status, and transfer ownership in a **trustless and immutable** environment.

Traditional warranty systems are prone to loss, fraud, and manual verification delays. This project solves those issues by storing warranty data on the blockchain, ensuring transparency, security, and easy access.

---

## 💡 What It Does

- 📦 Register products with warranty details on-chain  
- 🔍 Check if a product warranty is valid  
- 🔄 Transfer product ownership securely  
- 📜 Store warranty records immutably on blockchain  

---

## ✨ Features

- ⚡ Fast Transactions (~5s finality)
- 💰 Low Cost (< $0.01 per transaction)
- 🔐 Tamper-proof records
- 🌐 Fully decentralized (permissionless)
- 🔎 Instant warranty verification
- 🔁 Ownership transfer support
- 🧩 Built using Soroban Smart Contracts (Stellar)

---

## 🖥️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Blockchain: Stellar (Soroban Smart Contracts)  
- Deployment: Stellar Testnet  
- Wallet Integration: Freighter / Stellar Wallet
